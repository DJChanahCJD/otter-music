# App Assets

Place your icon and splash screen source files here.

## Requirements
- `icon.png`: App icon (1024x1024 px recommended)
- `splash.png`: Splash screen (2732x2732 px recommended)

## Optional (for Android Adaptive Icons)
- `icon-foreground.png`: Foreground layer
- `icon-background.png`: Background layer

## How to generate
Run the following command in the project root:
```bash
npm run resources
```
