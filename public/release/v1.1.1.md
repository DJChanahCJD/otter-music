# v1.1.1 Release Notes

**主要更新**
本次更新引入了 React Router 进行路由管理，重构了页面结构，并修复了移动端菜单和导航相关的多个问题。

## 新特性 (Features)
- **路由重构**: 引入 React Router 实现路由导航，优化页面结构 (a254ff9)

## 问题修复 (Bug Fixes)
- **菜单交互**: 修复移动端菜单项点击失效及队列变更触发重播的问题 (d863b4a)
- **下载优化**: 增加 Android 存储权限检查，防止下载崩溃 (56a4c39)

## 代码优化 (Refactoring)
- **组件简化**: 简化音乐组件接口并移除未使用的属性 (8c32f56)
- **代码清理**: 重命名喜欢相关函数并优化移除操作 (2bc5a2d)

## 其他 (Chore/Docs)
- 移除 .trae 目录的 gitignore 条目并添加项目文档 (21bbe06)
- 停止追踪 tsconfig.tsbuildinfo 文件 (16e8572)
- 更新项目图标并完善 README 文档 (ac064ee, c871b04)
