# v1.2.0 Release Notes

**主要更新**
本次更新带来了应用自动更新、歌单导入导出及去重功能，并重构了设置页面与收藏页面，同时修复了播放控制及同步相关的多个问题。

## 新特性 (Features)
- **版本更新**: 添加检查更新功能，支持检查新版本并下载 (a5cf99e)
- **歌单管理**: 新增歌单导入导出功能，方便数据迁移与备份 (261400e)
- **歌单去重**: 实现歌单去重功能，一键清理重复曲目 (c4fdc81)
- **收藏页面**: 新增独立的收藏页面组件，优化收藏管理体验 (e31d55d)
- **搜索增强**: 为歌曲下拉菜单添加歌手和专辑搜索功能，快速查找相关内容 (83f1787)

## 优化 (Improvements)
- **设置重构**: 重构设置页面，使用统一的 SettingItem 组件，提升界面一致性 (261400e, 830700a)
- **界面优化**: 优化播放列表组件图标及页面标题样式 (c59711b)
- **封面调整**: 将默认封面图片大小调整为 800px，提升视觉效果 (51327e2)

## 问题修复 (Bug Fixes)
- **同步提示**: 修复同步后提示及 API 域名更新问题 (8f77c5c)
- **播放控制**: 防止相同音轨重复加载及音质切换时触发暂停的问题 (d291212)
- **进度条**: 修复进度条显示问题，将 duration 添加到持久化状态 (80e71f3)
- **下载修复**: 修复低版本安卓无法使用下载功能的 bug (095e4c7)
