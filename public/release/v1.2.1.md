# v1.2.1 Release Notes

**主要更新**
本次更新重点提升了搜索体验（支持搜索意图与更准确的排序/提示），增强同步数据的健壮性，并优化导出提示与部分组件复用。

## 新特性 (Features)
- **搜索意图**: 支持按「歌手 / 专辑」的搜索意图发起搜索，提升搜索准确性 (069b3fc)

## 优化 (Improvements)
- **搜索排序**: 专辑/歌手搜索时优先展示“精确匹配”的结果（同名专辑/同名歌手优先） (069b3fc)
- **结果提示**: 搜索无结果时展示更明确的提示文案 (069b3fc)
- **导出提示**: 简化导出成功提示并缩短展示时长 (9dd71b6)
- **Toast 视觉**: 新增 notFound 提示，info/warning 图标改为更轻量的 Unicode 表情符号 (1dd9982)

## 问题修复 (Bug Fixes)
- **同步稳定性**: 同步前后增加数据验证与清理，防止无效数据导致同步失败或污染本地状态 (5e20323)
- **版本号显示**: 移除版本号中的 "v" 前缀，保持一致性 (9dd71b6)

## 代码优化 (Refactoring)
- **歌单菜单**: 提取歌单操作菜单为可复用组件，减少重复逻辑并统一样式 (3b045c8)
